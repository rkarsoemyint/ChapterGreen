# ChapterGreen - Online Bookstore & Admin Management System
ChapterGreen is a responsive Full-Stack Web Application designed for book enthusiasts to discover, save, and purchase their favorite books. It features an intuitive front-end user experience coupled with a robust Admin Dashboard for seamless inventory and order management.

# Features
# User Side
Dynamic Book Exploring: Browse books by category or utilize the dynamic, smooth-animating search layout to find titles instantly.
Shopping Cart & Wishlist: Add desired books to the shopping cart or curate a personalized wishlist in the user's collection.
Order Tracking: Track purchased items with updated status metrics in real-time.
Secure Checkout Simulation: Provides realistic payment flows for both Cash on Delivery (COD) and Online Card Payment simulations.

# Admin Side (Control Center)
Books Inventory Management: Full CRUD operations allowing administrators to seamlessly add, edit, or remove books from the library.
Order & Workflow Control: Monitor inbound orders and update fulfillment stages across Pending, Shipped, Delivered, or Cancelled statuses.
Store Revenue Analytics: An integrated analytics dashboard displaying real-time metrics for total revenue and overall order volume.

# Tech Stack
  - Frontend:
    React.js (Vite)
    Tailwind CSS (Responsive Design Architecture)
    Lucide React (Icon System)
    React Router DOM (Client-Side Routing)

 - Backend & Database:

    Node.js (Express.js)
    MongoDB (Mongoose ODM)

# Getting Started
Follow these steps to set up and run the project locally on your machine:

1. Clone the Repository    
   Bash

git clone https://github.com/rkarsoemyint/ChapterGreen.gitcd ChapterGreen

2. Install Dependencies
   Bash

   npm install
3. Run the Development Server
   Bash

   npm run dev
Open your browser and navigate to the local environment path (typically http://localhost:5173).
